# VR-Симулятор на основе игрового движка Unity для интерактивного моделирования поселения
Проект разработан на игровом движке Unity с использованием XR Interaction Toolkit и OpenXR.

## Назначение

Симулятор предназначен для релаксации и казуального проектирования в виртуальной реальности. Пользователь может захватывать миниатюрные модели зданий лучом контроллера, размещать их на виртуальном столе с автоматической привязкой к сетке, вращать и масштабировать объекты.

## Технологии

- **Unity** 6000.0.23f1 (LTS)
- **XR Interaction Toolkit** (для взаимодействия с VR-контроллерами)
- **OpenXR Plugin** (стандарт для кроссплатформенной VR-разработки)
- **Universal Render Pipeline (URP)** (оптимизация графики)
- **Язык C#**

## Целевая платформа

- Meta Quest 2 / 3 / Pro

## Структура репозитория

├── XRDragPreviewSnap.cs # Управление превью, привязка к сетке, проверка коллизий

├── RightTriggerTeleportToPreview.cs # Телепортация объекта по нажатию правого триггера

├── ActivateIfGamecontrolActive.cs # Синхронизация интерфейса с активностью превью

├── SpawnFromList.cs # Спавн объектов в интерактивной области

├── MusicPlayer.cs # Циклическое воспроизведение фоновой музыки

├── VolumeSlider.cs # Регулировка громкости через Audio Mixer

├── LoadSceneButton.cs # Загрузка сцены по нажатию кнопки UI

├── QuitGame.cs # Выход из приложения

├── VRPauseMenu.cs # Система паузы по кнопке контроллера

## Системные требования

- Unity 6000.0.23f1 или новее
- Установленные пакеты: XR Interaction Toolkit, OpenXR Plugin, URP
- Meta Quest Link / ADB для сборки и развертывания
